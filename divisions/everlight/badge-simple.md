# Everlight Badge — Simple

## Description
Minimal monoline badge derived from the Everlight emblem. Single-line diamond with central beam.

## Construction
- Use 12×12 master grid.
- Outer diamond: align to grid corners.
- Central beam: 1 unit width, centered.
- No fills.

## Stroke
- 2px monoline.

## Export
- SVG (monochrome)
- PNG 1x, 2x
