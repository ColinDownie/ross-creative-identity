# Everlight Badge — Ornate

## Description
Ornate variant for ceremonial use. Diamond with four subtle radiance ticks and thin inner inset.

## Construction
- Outer diamond on 12×12 grid.
- Inner inset diamond at 10% inset.
- Radiance ticks: 4 marks at 30° from central beam, 1 unit length.

## Stroke
- Primary 2px; inner inset 1px.

## Usage
- Print certificates, header art, ceremonial documents.

## Export
- SVG (layered)
- PDF (vector)
